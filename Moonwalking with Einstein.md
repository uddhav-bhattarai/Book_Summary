## Chapter 2: The Man Who Remembered Too Much

### Summary
Explores the case of Solomon Shereshevsky ("S"), studied by Russian psychologist Alexander Luria. S had virtually unlimited memory due to extreme synesthesia but struggled with normal comprehension and relationships.

### Key Insights
- **Synesthesia**: S experienced overwhelming sensory crossover (sounds had colors, numbers had textures)
- Perfect memory created problems: couldn't forget, struggled with metaphors and abstract concepts
- Demonstrates that forgetting is essential for normal cognitive function

## Chapter 5: The Memory Palace

### Summary
Foer begins training with Ed Cooke, learning the method of loci (memory palace technique). This ancient method uses spatial memory to store information.

### The Method of Loci Explained

**How It Works:**
1. Choose a familiar location (your house, commute route)
2. Create vivid, bizarre images for items to remember
3. Place images at specific locations along a mental journey
4. Recall by mentally walking through the space

### Rules for Effective Images
- **Bizarre and vivid**: Ordinary doesn't stick

## Key Techniques Summary

### 1. Memory Palace (Method of Loci)
- Use familiar spatial routes
- Place vivid images at locations
- Walk through mentally to recall

### 2. PAO System (Person-Action-Object)
- Encode numbers as images
- Combine three elements per image
- Reduces cognitive load dramatically

### 3. Major System
- Converts numbers to consonant sounds
- Sounds become words
- Words become memorable images

```
NUMBER TO SOUND CONVERSION:
0 = s, z (Zero)
1 = t, d (one downstroke)
2 = n (two downstrokes)
3 = m (three downstrokes)
4 = r (fouR)
5 = l (L is Roman 50)
6 = j, sh, ch
7 = k, hard g
8 = f, v
9 = p, b

Example: 32 = m-n = "moon"
         42 = r-n = "rain"
```

### 4. Linking Method
- Create story connecting items
- Each item triggers next
- Use bizarre, vivid imagery

### 5. Chunking
- Group information into meaningful units
- Reduce number of items to remember
- Build on existing knowledge
